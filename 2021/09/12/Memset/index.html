Memset
event 发布于: 2021-09-12 event_available 更新于: 2023-07-22 folder 分类于: 未分类 visibility 阅读次数: 

解释：
void *memset(void *str,int c,size_t n) 通过参数str将字符c（无符号字符）复制到所指向的字符串的前n个字符中。
       参数：
            str -指向要填充的字符串指针
      c -这是要设置的值。该值以int形式传递，但是该函数使用此值的无符号char转换填充内存块。
      n -要设置的字节数
返回：
该函数返回一个指向存储区str的指针。
在设置size_t n时要特别主要大小检测n的合法性，否则很容易溢出，使得破坏邻近内存空间里的数据。

输出

重置把附近内存的数据破坏了。
